Complete User Guide

Getting Facebook Cookie

  1. Login to Facebook in your browser
  2. Open Developer Tools (F12)
  3. Go to Application/Storage tab
  4. Find Cookies section
  5. Copy the value of the "c_user" and "xs" cookies
  6. Format: c_user=XXXX; xs=XXXX

Finding Post Links

Copy the URL from any Facebook post you want to boost.

Settings Explanation

  • Delay: Time between each share request (recommended: 1-100ms)
  • Limit: Maximum number of shares to attempt
  • Cookie: Your Facebook authentication cookie

Best Practices

  • Start with small limits (10-50) to test
  • Use delays to avoid detection
  • Monitor the process tab for real-time updates
  • Clear history regularly
